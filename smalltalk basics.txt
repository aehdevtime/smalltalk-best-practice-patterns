SELECTOR (method name)

A Smalltalk message is a request to an object instance to perform an operation.
A message expression consists of a receiver, selector (messagename), and potentially some arguments.

RETURN (^)

“^” indicates that an answer is returned

PERFORM (Equivalent to Ruby send())

DO

do: For each element of the collection do something (Smalltalk: collection do block. Ruby: collection.each do block).
